# SCT_DS_2

## Task
Perform Data Cleaning and Exploratory Data Analysis (EDA) on the Titanic Dataset.

## Dataset
Titanic Dataset from Kaggle.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Google Colab

## Data Cleaning
- Filled missing values in Age column.
- Filled missing values in Embarked column.
- Removed Cabin column due to excessive missing values.

## Exploratory Data Analysis
Created visualizations for:
1. Survival Count
2. Gender Distribution
3. Survival by Gender
4. Passenger Class vs Survival
5. Age Distribution
6. Fare Distribution

## Key Findings
- Female passengers had higher survival rates.
- First-class passengers survived more often than third-class passengers.
- Most passengers were between 20 and 40 years old.
- Fare distribution was highly skewed.

## Outcome
Learned data cleaning techniques and exploratory data analysis using Python.
